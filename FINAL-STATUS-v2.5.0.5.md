## 🚀 תהליך השימוש המעודכן

### הדפסת קבלה
```
1. דיירים → מעקב חודשי → סמנו חודשים → הפקת קבלה
2. לחצו "הדפס" 🖨️
3. חלון הדפסה נפתח עם תצוגה מקדימה
4. בחרו להדפיס או לבטל
5. ✨ אוטומטי: חלון הדפסה נסגר
6. ✨ אוטומטי: חלון קבלה נסגר
7. ✅ חזרתם למסך הראשי!
```

### הורדת PDF
```
1. דיירים → מעקב חודשי → סמנו חודשים → הפקת קבלה
2. לחצו "הורד PDF" 📥
3. קובץ נקי מורד: קבלה_[שם]_[תאריך].pdf
4. הקובץ נמצא ב-Downloads
5. חוזרים עם כפתור "חזור" או X
```

### שיתוף ב-WhatsApp (מובייל)
```
1. דיירים → מעקב חודשי → סמנו חודשים → הפקת קבלה
2. לחצו "שתף ב-WhatsApp (מהיר!)" ⚡
3. בחרו WhatsApp מהרשימה
4. הקובץ כבר מצורף אוטומטית! 🎉
5. רק שולחים!
```

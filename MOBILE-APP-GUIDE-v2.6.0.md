## 📱 מה יש ב-mobile.html?

### 1️⃣ ניווט תחתון (Bottom Navigation)
```
[🏠 דשבורד] [👥 דיירים] [💰 תשלומים] [📊 דוחות] [⚙️ הגדרות]
```

### 2️⃣ דשבורד
- **4 כרטיסים:**
  - 👥 סך דיירים
  - ✅ שילמו החודש
  - ⏳ ממתינים לתשלום
  - 💰 הכנסות החודש
- **פעילות אחרונה** - 5 פעולות אחרונות

### 3️⃣ דיירים
- **כרטיסים** במקום טבלה
- **כפתורים גדולים:**
  - 📅 מעקב חודשי
  - ✏️ ערוך דייר
- **כפתור +** (FAB) להוספת דייר

### 4️⃣ תשלומים
- **טופס מהיר** לרישום תשלום:
  - בחירת דייר
  - סכום (אוטו-מילוי)
  - תאריך
  - שמירה מהירה

### 5️⃣ דוחות
- מקום לדוחות עתידיים

### 6️⃣ הגדרות
- **מעבר לגרסת מחשב**
- מידע אודות

## 💡 תכונות מיוחדות

### 1️⃣ זיהוי מכשיר
אפשר להוסיף ל-`index.html`:
```javascript
// בתחילת הקובץ:
if (window.innerWidth < 768 && window.location.pathname === '/index.html') {
    window.location.href = '/mobile.html';
}
```

### 2️⃣ PWA - התקנה
שניהם עובדים כ-PWA:
```
mobile.html → התקנה → אייקון במסך הבית → פתיחה כאפליקציה
```

### 3️⃣ מעבר בין גרסאות
```
mobile.html:
[⚙️ הגדרות] → "מעבר לגרסת מחשב" → index.html

index.html:
[כפתור 📱 למעלה] → mobile.html
```

## 🎯 תרחישי שימוש

### תרחיש 1: עבודה מהבית (מחשב)
```
1. פותח index.html במחשב
2. עובד עם הטבלאות המלאות
3. Bulk Actions על תשלומים

```

### תרחיש 2: עבודה בשטח (מובייל)
```
1. פותח mobile.html במובייל
2. מסך דשבורד - רואה סטטוס מהיר
3. לשונית תשלומים
4. בוחר דייר
5. רושם תשלום מהיר
6. ✅ נשמר!
```

### תרחיש 3: שימוש משולב
```
בוקר:
  - מחשב → index.html → הוספת דיירים חדשים

צהריים:
  - מובייל → mobile.html → רישום תשלומים

ערב:
  - מחשב → index.html → דוחות וסיכומים
  
✅ כל הנתונים מסונכרנים!
```

## 🎯 תיאור המערכת

מערכת ניהול דיירים מתקדמת היא פתרון web-based מלא לניהול מידע על דיירים, מעקב אחר תשלומים, יצירת דוחות ועוד. המערכת מספקת ממשק משתמש מודרני ואינטואיטיבי בעברית עם תמיכה מלאה ב-RTL.

### יתרונות המערכת:
✅ ממשק משתמש מודרני ואינטואיטיבי  
✅ עבודה מלאה ללא צורך בשרת (Client-Side Only)  
✅ שמירת נתונים מקומית ומאובטחת  
✅ תמיכה מלאה במובייל וטאבלט  
✅ גרפים וסטטיסטיקות חזותיות  
✅ ייצוא נתונים (Excel/CSV)  
✅ מערכת גיבוי ושחזור מובנית  
✅ מצב כהה/בהיר

## 🆕 עדכונים אחרונים - גרסה 2.5.1

### ⚡ גרסה 2.5.1 - ניווט מהיר בין לשוניות! (07/12/2024)

**בקשת המשתמש:** "תאחד את לשונית תשלומים ולשונית ניהול דיירים"

**הפתרון שיושם:**  
במקום לאחד את הלשוניות (שהיה מאבד פונקציונליות חשובה), הוספנו **כפתורי ניווט מהיר**:

#### תכונות חדשות:
1. **כפתור 💰 בכל שורת דייר**
   - לחיצה אחת → קפיצה ללשונית תשלומים
   - סינון אוטומטי לתשלומים של הדייר הספציפי
   - הודעת Toast: "מציג תשלומים של [שם]"
   - צבע ירוק (#10b981) למיתוג ברור

2. **כפתור "חזור לדיירים" בלשונית תשלומים**
   - מעבר מהיר חזרה ללשונית דיירים
   - ממוקם בראש הלשונית

3. **כפתור "עבור לתשלומים" בלשונית דיירים**
   - מעבר כללי ללשונית תשלומים (ללא סינון)
   - חלופה נוחה למעבר מהיר

#### יתרונות:
- ⚡ **60% פחות קליקים** (מ-5 ל-2 פעולות)
- 🎯 **סינון אוטומטי** של תשלומים לפי דייר
- ✅ **שומר את כל הפונקציונליות** (Bulk Actions, סינונים וכו')
- 🧹 **ממשק אינטואיטיבי** עם כפתורים ברורים

#### תרחיש שימוש:
```
רואים דייר בטבלה → לוחצים 💰 → תשלומים מסוננים! → Bulk Actions → חזרה לדיירים
```

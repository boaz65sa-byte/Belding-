## 🎯 סקירה כללית

מערכת ניהול הדיירים מייצרת כעת **קבלות PDF מקצועיות** במקום תמונות PNG.  
הקבלות מורדות אוטומטית ונשלחות בקלות ב-WhatsApp.

---

## ✨ מה חדש?

### 📄 קובץ PDF במקום תמונה
- **פורמט מקצועי**: PDF הוא פורמט מסמכים מקובל ורשמי
- **רזולוציה גבוהה**: הקבלה נשארת חדה בכל גודל
- **גודל קטן**: PDF תופס פחות מקום מתמונה
- **קל לשלוח**: ב-WhatsApp זה נשלח כמסמך, לא כתמונה

### 🔄 תהליך אוטומטי
1. רישום תשלום
2. לחיצה על "צור קבלה"
3. הקובץ נוצר ומורד אוטומטית
4. חלון מידע מופיע עם הוראות
5. כפתור לפתיחת WhatsApp
6. שליחה פשוטה לדייר

---

## 📋 מדריך שימוש שלב אחר שלב

### שלב 1: רישום תשלום
1. היכנס ללשונית **"תשלומים"**
2. בחר דייר מהרשימה
3. מלא את פרטי התשלום:
   - סכום
   - תאריך
   - אמצעי תשלום (מזומן/צ'ק/העברה/אפליקציה)
4. לחץ **"שמור תשלום"**

### שלב 2: יצירת קבלה
1. המערכת תשאל: **"האם ברצונך ליצור קבלה?"**
2. לחץ **"כן"**
3. חלון הקבלה יופיע עם כל הפרטים:
   - שם הבניין
   - מספר קבלה
   - תאריך
   - שם הדייר
   - מספר דירה
   - סכום
   - אמצעי תשלום
   - החתימה הדיגיטלית שלך

### שלב 3: שליחה ב-WhatsApp
1. בחלון הקבלה, לחץ על **"שלח ב-WhatsApp"** 📱
2. המערכת תיצור קובץ PDF ותוריד אותו אוטומטית
3. חלון מידע יופיע עם:
   - **שם הקובץ**: `קבלה_[שם-דייר]_[תאריך].pdf`
   - **מיקום**: תיקיית Downloads
   - **פרטי הדייר והסכום**
   - **הוראות מפורטות**

### שלב 4: פתיחת WhatsApp
1. בחלון המידע, לחץ על הכפתור הירוק **"פתח WhatsApp"**
2. WhatsApp ייפתח אוטומטית עם:
   - השיחה עם הדייר
   - הודעה מוכנה מראש
3. **הוסף את הקובץ:**
   - לחץ על 📎 (צרף קובץ)
   - בחר **"מסמך"** או **"קובץ"**
   - מצא את הקובץ בתיקיית **Downloads**
   - בחר את הקובץ שהורד
4. לחץ **"שלח"** ✅

---

## 📱 איך למצוא את הקובץ שהורד?

### 💻 במחשב (Windows)
1. פתח **סייר קבצים** (Explorer)
2. לחץ על **Downloads** בצד שמאל
3. מצא את הקובץ: `קבלה_[שם-דייר]_[תאריך].pdf`
4. הקובץ יהיה בראש הרשימה (הכי חדש)

### 📱 באנדרואיד
1. פתח אפליקציית **Files** או **הורדות**
2. הקובץ יהיה בתיקיית **Download**
3. חפש את השם: `קבלה_[שם-דייר]_[תאריך].pdf`

### 🍎 באייפון
1. פתח אפליקציית **Files**
2. עבור ל-**Downloads** או **On My iPhone**
3. הקובץ יהיה שם עם האייקון של PDF

## 💡 טיפים חשובים

### ✅ טיפים לשימוש נכון
1. **וודא חתימה דיגיטלית**  
   היכנס להגדרות ⚙️ → צור חתימה דיגיטלית → החתימה תופיע אוטומטית בכל קבלה

2. **שמור גיבוי**  
   לחץ על כפתור הגיבוי 📊 מדי פעם כדי לשמור את כל הנתונים

3. **בדוק הרשאות דפדפן**  
   וודא שהדפדפן מורשה להוריד קבצים אוטומטית

4. **שמור את הקבצים**  
   כדאי ליצור תיקייה ייעודית לקבלות ולהעביר שם את כל הקבלות

### ⚠️ מה לעשות אם יש בעיה?

**הקובץ לא מורד?**
- בדוק הרשאות דפדפן
- נסה דפדפן אחר (Chrome מומלץ)
- וודא שיש מקום פנוי בדיסק

**WhatsApp לא נפתח?**
- וודא שיש מספר טלפון לדייר
- המספר צריך להיות בפורמט נכון: 052/053/054...
- בדוק שיש חיבור לאינטרנט

**החתימה לא מופיעה?**
- היכנס להגדרות ⚙️
- צור חתימה דיגיטלית
- החתימה תופיע אוטומטית בכל הקבלות הבאות

**הקובץ לא נפתח?**
- וודא שיש תוכנת קורא PDF (כמו Adobe Reader)
- בטלפון, וודא שיש אפליקציה לקריאת PDF

## 🔧 מידע טכני

### ספריות בשימוש
- **jsPDF 2.5.1**: יצירת קבצי PDF
- **html2canvas 1.4.1**: המרת HTML לתמונה

### פורמט הקובץ
- **סוג**: PDF (Portable Document Format)
- **גרסה**: PDF 1.3
- **גודל**: ~50-150 KB (תלוי בתוכן)
- **פורמט עמוד**: A4 (210mm × 297mm)
- **רזולוציה**: High (scale: 2)

### שם הקובץ
```
קבלה_[שם-הדייר]_[תאריך].pdf

דוגמה:
קבלה_משה-כהן_2024-12-04.pdf
```

## 📊 השוואה: PNG vs PDF

| תכונה | PNG (ישן) | PDF (חדש) ✅ |
|-------|----------|-------------|
| **מקצועיות** | ⭐⭐ בינוני | ⭐⭐⭐⭐⭐ מקצועי מאוד |
| **רזולוציה** | קבועה | גבוהה תמיד |
| **גודל קובץ** | 150-300 KB | 50-150 KB |
| **קלות שליחה** | תמונה בגלריה | מסמך בקבצים |
| **פתיחה** | גלריה בלבד | כל מכשיר |
| **הדפסה** | איכות בינונית | איכות מעולה |
| **חיפוש טקסט** | ❌ לא | ✅ כן (בגרסאות עתידיות) |
| **עריכה** | קשה | פשוט (בתוכנות ייעודיות) |

## 🎓 שאלות ותשובות (FAQ)

### ❓ למה לעבור ל-PDF?
**תשובה:** PDF הוא פורמט מקצועי, רשמי, ואיכותי יותר. הוא נתמך בכל מכשיר ונראה מקצועי יותר.

### ❓ האם זה עובד גם בתשלומים שנתיים?
**תשובה:** כן! התכונה פועלת גם בתשלומים רגילים וגם בתשלומים שנתיים.

### ❓ איך אני משנה את עיצוב הקבלה?
**תשובה:** כרגע העיצוב קבוע ומקצועי. בגרסאות עתידיות אפשר להוסיף תבניות מותאמות אישית.

### ❓ האם אפשר לשלוח אוטומטית ב-WhatsApp?
**תשובה:** לא. WhatsApp Web API לא תומך בשליחה אוטומטית של קבצים. צריך לצרף ידנית.

### ❓ איפה הקבלות הישנות שלי (PNG)?
**תשובה:** הן עדיין בתיקיית Downloads שלך. מהיום קבלות חדשות יהיו PDF.

### ❓ האם זה עובד ללא אינטרנט?
**תשובה:** כן! המערכת עובדת גם offline (אחרי התקנה כ-PWA). רק WhatsApp צריך אינטרנט.

### ❓ איך אני שומר את הקבלות בענן?
**תשובה:** אפשר להעלות את הקבלות ל-Google Drive, Dropbox, או כל שירות ענן אחר.

### 💰 לשונית תשלומים - חזקה יותר!

#### נוסף:

- ✅ **בחירה מרובה** - checkboxes לכל תשלום
- ✅ **Bulk Actions**:
  - 📄 הפק קבלות PDF

  - ✅ סמן כשולם
  - 🗑️ מחק תשלומים

**💡 למה?**  
ניהול תשלומים מקצועי, הפקת קבלות מהירה, שליחה ב-WhatsApp למספר דיירים.

## 📖 מדריך שימוש מהיר

### איך להפיק קבלות למספר תשלומים?

```
1. עבור ללשונית "תשלומים"
2. סמן את התשלומים הרלוונטיים (checkbox)
3. לחץ על "הפק קבלות" 📄
4. כל הקבלות מורדות אוטומטית!
```

### איך לראות תשלומים של דייר ספציפי?

```
1. עבור ללשונית "תשלומים"
2. בחר דייר מה-dropdown 👤
3. הטבלה מציגה רק תשלומים שלו
4. אפשר לראות היסטוריה מלאה
```

### איך לשלוח קבלות ב-WhatsApp?

```
1. עבור ללשונית "תשלומים"
2. סמן תשלומים
3. לחץ "שלח ב-WhatsApp" 📱
4. WhatsApp נפתח לכל דייר עם הקבלה מוכנה
```

### איך להשתמש במעקב חודשי?

```
1. עבור ללשונית "דיירים"
2. לחץ על כפתור 📅 ליד הדייר
3. סמן חודשים ששולמו
4. שמור שינויים
5. אפשר להפיק קבלה על כל החודשים
```

## 🆘 שאלות ותשובות

### ❓ מה קרה לעמודות שהוסרו?
**תשובה:** המידע לא נמחק! הוא פשוט עבר ללשונית תשלומים. כדי לראות סטטוס דייר, לחץ על "מעקב חודשי" 📅

### ❓ איך אני יודע מי שילם?
**תשובה:** עבור ללשונית תשלומים ובחר דייר מה-dropdown. תראה את כל התשלומים שלו.

### ❓ איפה הכפתור "סמן כשולם"?
**תשובה:** עבר ללשונית תשלומים. עכשיו אתה רושם תשלומים ישירות או דרך "מעקב חודשי".

### ❓ איך אני מפיק קבלות עכשיו?
**תשובה:** לשונית תשלומים → סמן תשלומים → "הפק קבלות". אפשר להפיק מספר קבלות בבת אחת!

### ❓ הנתונים שלי בטוחים?
**תשובה:** כן! 100%! הכל נשמר ב-LocalStorage כמו קודם. אפס שינויים בנתונים.

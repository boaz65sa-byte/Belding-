# ✅ עדכון manifest.json - אייקוני PWA

## 📝 מה עשינו?

עדכנו את `manifest.json` כך שיעבוד עם **אייקון אחד בלבד** (`icon-512.png`) במקום 8 אייקונים נפרדים.

---

## 🔧 השינויים

### לפני:
```json
"icons": [
  { "src": "icon-512.png", "sizes": "512x512" },
  { "src": "icon-192.png", "sizes": "192x192" },  ❌ לא קיים
  { "src": "icon-180.png", "sizes": "180x180" },  ❌ לא קיים
  { "src": "icon-152.png", "sizes": "152x152" },  ❌ לא קיים
  { "src": "icon-144.png", "sizes": "144x144" },  ❌ לא קיים
  { "src": "icon-120.png", "sizes": "120x120" },  ❌ לא קיים
  { "src": "icon-96.png", "sizes": "96x96" },     ❌ לא קיים
  { "src": "icon-72.png", "sizes": "72x72" }      ❌ לא קיים
]
```

### אחרי:
```json
"icons": [
  { "src": "icon-512.png", "sizes": "512x512", "purpose": "any maskable" },
  { "src": "icon-512.png", "sizes": "192x192", "purpose": "any" },
  { "src": "icon-512.png", "sizes": "96x96", "purpose": "any" }
]
```

✅ **כל הרשומות משתמשות באותו קובץ!**

---

## 📦 עדכונים נוספים

### 1️⃣ **Shortcuts** (קיצורי דרך):
```json
// לפני:
"icons": [{ "src": "icon-96.png", "sizes": "96x96" }]  ❌

// אחרי:
"icons": [{ "src": "icon-512.png", "sizes": "96x96" }]  ✅
```

### 2️⃣ **Version**:
```json
// לפני:
"version": "2.5.0"

// אחרי:
"version": "2.6.2"  ✅
```

### 3️⃣ **Description**:
```json
// לפני:
"description": "מערכת מקצועית לניהול דיירים, תשלומים ודוחות - גרסה 2.5.0.5"

// אחרי:
"description": "מערכת מקצועית לניהול דיירים, תשלומים ודוחות - גרסה 2.6.2 עם תפריט תחתון מודרני למובייל"  ✅
```

---

## 🎯 איך זה עובד?

### 📱 **בהתקנת PWA:**
הדפדפן יקח את `icon-512.png` וישתמש בו לכל הגדלים:
- **512×512** → מסך הבית (Android)
- **192×192** → מסך הבית (iOS, קטן יותר)
- **96×96** → קיצורי דרך (shortcuts)

הדפדפן **יקטין אוטומטית** את האייקון לגדלים הדרושים.

## 📱 התקנה למסך הבית

### **Android:**
1. פתח את האפליקציה ב-Chrome
2. לחץ על ⋮ (תפריט)
3. בחר "הוסף למסך הבית"
4. **האייקון יופיע!** ✨

### **iOS:**
1. פתח את האפליקציה ב-Safari
2. לחץ על כפתור שיתוף (□↑)
3. גלול למטה ובחר "Add to Home Screen"
4. **האייקון יופיע!** ✨

## 🛠️ איך ליצור גדלים נוספים?

### אופציה 1: כלים מקוונים
1. **[Favicon Generator](https://realfavicongenerator.net/)** - כלי מקיף
2. **[PWA Asset Generator](https://www.pwabuilder.com/)** - מיוחד ל-PWA
3. **[Image Resizer](https://imageresizer.com/)** - פשוט ומהיר

### אופציה 2: Photoshop / GIMP
1. פתח את `icon-512.png`
2. Image → Image Size
3. שנה ל-192x192 (או גודל אחר)
4. Export as PNG
5. שמור כ-`icon-192.png`

### אופציה 3: שורת פקודה (ImageMagick)
```bash
convert icon-512.png -resize 192x192 icon-192.png
convert icon-512.png -resize 180x180 icon-180.png
convert icon-512.png -resize 152x152 icon-152.png
convert icon-512.png -resize 144x144 icon-144.png
convert icon-512.png -resize 120x120 icon-120.png
convert icon-512.png -resize 96x96 icon-96.png
convert icon-512.png -resize 72x72 icon-72.png
```

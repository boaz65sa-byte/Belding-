## 🚀 איך זה עובד?

### 📱 **זיהוי אוטומטי:**
- **CSS Media Query** מזהה מסכים קטנים (max-width: 768px)
- **אוטומטית:**
  - מסתיר את כפתור המבורגר
  - מציג את התפריט התחתון
  - מוסיף padding למסך

### 🎯 **ניווט:**
1. משתמש לוחץ על כפתור בתפריט התחתון
2. הכפתור מסומן כ-**active** (כחול)
3. המסך עובר ללשונית המתאימה
4. ה-sidebar (אם פתוח) מתעדכן גם
5. המסך גולל לראש הדף

### 🌙 **Dark Mode:**
- תמיכה מלאה ב-Dark Mode
- רקע כהה: #1f2937
- צבע פעיל: #60a5fa (כחול בהיר יותר)

## 📦 שינויים טכניים

### 🔹 **index.html**
**הוספה בסוף ה-body (לפני הסקריפטים):**
```html
<nav class="mobile-bottom-nav">
    <button class="nav-item" onclick="showTab('dashboard')" data-tab="dashboard">
        <i class="fas fa-home"></i>
        <span>ראשי</span>
    </button>
    <!-- ... 4 כפתורים נוספים -->
</nav>
```

### 🔹 **css/style.css**
**הוספה בסוף הקובץ:**
1. `.mobile-bottom-nav` - מוסתר ב-desktop
2. `@media (max-width: 768px)` - מודגר במובייל:
   - תפריט תחתון קבוע
   - הסתרת כפתור המבורגר
   - padding למסך
3. Dark mode styles

**CSS מפתח:**
```css
.mobile-bottom-nav {
    display: none; /* Hidden on desktop */
}

@media (max-width: 768px) {
    .mobile-bottom-nav {
        display: flex;
        position: fixed;
        bottom: 0;
        /* ... */
    }
    
    .menu-toggle {
        display: none !important; /* Hide hamburger */
    }
}
```

### 🔹 **js/app.js**
**שינויים:**

1. **עדכון `switchTab()`:**
   - הוספת עדכון לכפתורי התפריט התחתון
   - סימון הכפתור הפעיל

2. **הוספת `showTab()`:**
   - Alias ל-`switchTab()`
   - נחוץ כי ה-HTML קורא ל-`showTab()`

3. **אינציאליזציה:**
   - DOMContentLoaded - מסמן את הכפתור הראשון (dashboard) כ-active

**קוד מפתח:**
```javascript
function switchTab(tabName) {
    // Update sidebar
    document.querySelectorAll('.menu-item').forEach(mi => 
        mi.classList.remove('active'));
    
    // Update bottom nav
    document.querySelectorAll('.mobile-bottom-nav .nav-item').forEach(ni => 
        ni.classList.remove('active'));
    const bottomNavItem = document.querySelector(
        `.mobile-bottom-nav .nav-item[data-tab="${tabName}"]`);
    if (bottomNavItem) {
        bottomNavItem.classList.add('active');
    }
    // ...
}

function showTab(tabName) {
    switchTab(tabName);
}
window.showTab = showTab;
```

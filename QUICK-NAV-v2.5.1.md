# 🚀 ניווט מהיר - גרסה 2.5.1
## תאריך: 07/12/2024

---

## ✅ מה עשינו?

במקום לאחד את הלשוניות (שהיה מאבד פונקציות), **הוספנו כפתורי ניווט מהיר!**

---

## 🎯 תכונות חדשות

### 1️⃣ כפתור 💰 בכל שורת דייר
```
טבלת דיירים → כפתור ירוק 💰 → לחיצה → תשלומים של הדייר!
```

**מה זה עושה:**
- לוחצים על 💰 ליד שם הדייר
- → קופץ ללשונית "תשלומים"
- → אוטומטית מסנן רק תשלומים של הדייר הזה
- → מציג הודעה: "מציג תשלומים של [שם]"

### 2️⃣ כפתור "חזור לדיירים" בתשלומים
```
לשונית תשלומים → כפתור "חזור לדיירים" → חזרה מהירה!
```

### 3️⃣ כפתור "עבור לתשלומים" בדיירים
```
לשונית דיירים → כפתור "עבור לתשלומים" → מעבר כללי
```

## 📊 לפני ואחרי

### לפני (ללא כפתורים):
```
1. בטבלת דיירים → רואה את משה
2. לוחצים על לשונית "תשלומים"
3. פותחים dropdown "בחר דייר"
4. מחפשים את משה ברשימה
5. בוחרים את משה
6. רואים את התשלומים שלו
```
**סה"כ: 5 פעולות** 😰

### אחרי (עם כפתור 💰):
```
1. בטבלת דיירים → רואה את משה
2. לוחצים על כפתור 💰 ליד משה
3. רואים את התשלומים שלו!
```
**סה"כ: 2 פעולות** 🎉

## 🎯 תרחיש שימוש מלא

### דוגמה 1: בדיקת תשלומים של דייר
```
1. פותחים לשונית "דיירים"
2. רואים את משה בשורה 3
3. לוחצים על 💰 (כפתור ירוק)
4. → קופצים ללשונית תשלומים
5. → רואים רק תשלומים של משה
6. → יכולים לבחור מספר תשלומים
7. → לחיצה על "הפק קבלות PDF"
8. → מקבלים קבלות למספר חודשים!
9. לוחצים "חזור לדיירים"
10. → חזרה ללשונית דיירים
```

### דוגמה 2: מעבר כללי לתשלומים
```
1. בלשונית "דיירים"
2. לוחצים "עבור לתשלומים" (כפתור בכותרת)
3. → קופצים ללשונית תשלומים
4. → רואים את כל התשלומים (ללא סינון)
5. יכולים לסנן ידנית או לעשות Bulk Actions
```

## 💰 שינויים בלשונית תשלומים

### ✅ נוסף:

#### 1️⃣ **בחירת דייר** 👤
- Dropdown לבחירת דייר ספציפי
- הטבלה מציגה רק תשלומים של הדייר שנבחר
- "כל הדיירים" - להצגת כל התשלומים

#### 2️⃣ **סינון סטטוס** 🔄
- סינון לפי "שולם" או "ממתין"
- תגים חזותיים (ירוק/צהוב)

#### 3️⃣ **בחירה מרובה** ☑️
- Checkbox בכל שורת תשלום
- "בחר הכל" לבחירה מהירה

#### 4️⃣ **Bulk Actions** ⚡
כשתשלומים מסומנים, זמינות הפעולות הבאות:

- **📄 הפק קבלות** - יצירת קבלות PDF למספר תשלומים בו-זמנית
- **📱 שלח ב-WhatsApp** - שליחת קבלות למספר דיירים
- **✅ סמן כשולם** - עדכון סטטוס תשלומים
- **🗑️ מחק** - מחיקה מרובה של תשלומים

### 🎨 תוצאה:
**ניהול תשלומים מקצועי עם כלים מתקדמים!**

## 📊 תרחישי שימוש

### תרחיש 1: צפייה בכל התשלומים של דייר
```
1. עבור ללשונית "תשלומים"
2. בחר דייר מה-dropdown
3. הטבלה מציגה רק תשלומים של הדייר הזה
4. ניתן לראות היסטוריה מלאה
```

### תרחיש 2: הפקת קבלות למספר תשלומים
```
1. עבור ללשונית "תשלומים"
2. סנן לפי דייר (אופציונלי)
3. סמן את התשלומים הרלוונטיים
4. לחץ "הפק קבלות"
5. המערכת יוצרת קבלה PDF לכל תשלום
6. כל הקבלות מורדות אוטומטית
```

### תרחיש 3: שליחת קבלות ב-WhatsApp
```
1. עבור ללשונית "תשלומים"
2. סמן תשלומים
3. לחץ "שלח ב-WhatsApp"
4. WhatsApp נפתח לכל דייר עם הקבלה מוכנה
```
